# Project: Food Finder

This is a [Next.js](https://nextjs.org) project built using lessons from _The Complete Developer_, written by Martin Krause.

The project employs a full-stack demonstration of a simple restaurant-locator app that uses JWTs and REST calls.

## The stack:
* **Runtime:** Node.js
* **Language:** TypeScript (ES6+)
* **Database:** Mongodb / Mongoose
* **Application:** Next.js
* **Environment:** WSL2 (Ubuntu) / Docker
* **Testing:** Jest
