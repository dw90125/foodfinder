# Project: Food Finder

This is a [Next.js](https://nextjs.org) project built using lessons from _The Complete Developer_, written by Martin Krause.

The project employs a full-stack demonstration of a simple restaurant-locator app.

## The stack:
* **Runtime:** Node.js
* **Language:** TypeScript (ES6+)
* **Database:** Mongodb / Mongoose
* **Application:** Next.js / GraphQL
* **Environment:** WSL2 (Ubuntu) / Docker
* **Testing:** Jest

## Live Demo:

I've set up a live demo for this project using [Render](https://foodfinder-mkq5.onrender.com/) to host the Next.js application, and [MongoDB Atlas](https://www.mongodb.com/products/platform) to host the database.  Also featured is a "Wishlist" functionality powered by Github OAuth.
